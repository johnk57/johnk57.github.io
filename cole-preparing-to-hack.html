Warm-up:

  • Go to https://microsoft-edge-chromium-devtools.glitch.me/static/console/javascript/index.html


  • Before you do anything, notice what the button on the webpage says.


  • Open the developer tools / console of your browser. On Chrome browser, I think it's Ctrl-Shift-I.


  • Copy the following javascript into your console but don't press enter yet.
    • You copy it into the part at the bottom of the console tab where it has a little prompt that usually looks like this "> " with a cursor blinking after it.
document.getElementById('hello').textContent = 'Cole was here!'


  • Once you've copied that in, keep an eye on the button on the webpage ...as you press enter.
...you changed the button on that webpage!   ¯\(◉‿◉)/¯   Congrats!!


Ok you can close that page, that was the warm-up.




Start on HackTheBox:

  • Go to https://www.hackthebox.eu/ and click "Hack Your Way In"


  • See what you can find using the developer tools / console of your browser. Did you find the first hint?


  • Find a way to take notes somewhere because HackTheBox will take a few steps. So start a google doc or something, and keep track of stuff we try.
    • Start right at the beginning by listing something like "Step 1 - Used the console and found this:" and then copy and paste any hints or clues or interesting things you find.


  • Then try to look around and see if you can figure out the clue. (It's ok if you can't, but make sure to take a few minutes to look around, because sometimes you learn something by doing that!)


  • Now list the next step on your google doc or wherever you're taking notes. And if you took time to look around to figure out the clue but you don't know what to do next, still list it, but list it as a question. Something like, "Step 2 - It's telling me _________, but _________?"
    • So for example, if you didn't know how to do the first step, the one about the console, you'd type in "Step 1 - It's telling me to use the console, but what is the console and how do I use it in my browser?"


  • And then here's the big trick: Just take your question, whatever step you get stuck on, and google it.
    • So in that example about step 1, you'd google "what is the console and how do I use it in my browser?" and learn how to do that. Go ahead, try it with whatever question you put in your notes about the part you're now stuck on.


To hack, you just do that for every step: A) take time to look around and think about the clue, and B) if you don't know something,list it in the form of a question in your notes, then B) google that question. Use that process to see if you can get through 1 or 2 or even 3 steps. Even if you only start step 1, that's great.


Whenever you're completely stuck, save your notes because that's where we'll start: We'll start with your question. And I promise we'll hack all the way in!! For now, move on to the last thing to prep.



Access learning games:

  • Sign up for an account at https://picoctf.org/


  • It should log you into https://play.picoctf.org/practice which has a bunch of puzzles to learn hacking, so bookmark that page.
    • It's a type of game called "Capture The Flag" or CTF and people run other CTF's for professional hackers that have cash prize$$$$$.


  • Whenever you want to read some hacking tricks, up at the top choose Learn > Resources, and learn the stuff there. Bookmark that page too: https://picoctf.org/resources


  • Next a cool part, open another tab and use the same username and password to log in to https://webshell.picoctf.org/


  • That webshell gives you a free Linux terminal to use! So bookmark that one too.


  • Try out some Linux commands in it:

clear

date

ls  (which stands for list)

history  (shows you all the commands you tried in order)

touch Cole-file.txt  (creates an empty file by that name)

ls  (to list what's in the directory again ...and you can see your new file!)

exit



If we have time after we hack into the HackTheBox site, we can come back to this Linux terminal and learn how to put stuff in your file. We can also try some of the CTF challenges if you want.
